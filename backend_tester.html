Backend API Tester

Test the Course Equivalency Finder backend API endpoints. Make sure your Flask server is running on localhost:5000

Basic Data Endpoints

GET /institutions

Get all institutions

GET /departments

Get departments for an institution

GET /courses

Get courses for a department

GET /equivalents

Get equivalent courses for a course

Program Endpoints

GET /programs

Get programs for a department

GET /programs/{id}/requirements

Get program with requirements

POST /programs/{id}/analyze

Analyze program completion

POST /programs

Create a new program

Transfer Plan Endpoints

POST /create-plan

Create a transfer plan

GET /get-plan/{code}

Get a transfer plan by code

DELETE /delete-plan/{code}

Delete a transfer plan

Import Endpoint

POST /import

Import course data from CSV

POST /programs/import

Import program data from CSV
