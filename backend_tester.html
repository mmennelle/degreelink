Backend API Tester

Test the Course Equivalency Finder backend API endpoints. Make sure your Flask server is running on localhost:5000

Course Endpoints

GET /courses

Get courses with search filters

GET /courses/{id}

Get a specific course with equivalencies

POST /courses

Create a new course

Equivalency Endpoints

GET /equivalencies

Get course equivalencies with filters

POST /equivalencies

Create a new equivalency

Program Endpoints

GET /programs

Get programs for a department

GET /programs/{id}/requirements

Get program with requirements

POST /programs/{id}/analyze

Analyze program completion

POST /programs

Create a new program

Plan Endpoints

GET /plans

Get plans with optional filters

POST /plans

Create a new plan

GET /plans/{id}

Get a specific plan with progress analysis

POST /plans/{id}/courses

Add a course to a plan

Upload Endpoints

POST /upload/courses

Upload courses from CSV

POST /upload/equivalencies

Upload equivalencies from CSV

POST /upload/requirements

Upload program requirements from CSV
